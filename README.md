# 📝 Feedback Management System

A full-stack Feedback Management System built using the MERN Stack. This application allows users to submit feedback and receive responses from administrators through a clean and user-friendly interface.

## 🚀 Features

### 👤 User Features
- User Registration
- User Login
- Submit Feedback/Suggestions
- View Submitted Feedback
- View Admin Replies

### 👮 Admin Features
- Admin Login
- View All User Feedback
- Reply to Feedback
- Manage User Suggestions Efficiently

## 🛠️ Tech Stack

### Frontend
- React.js
- Vite
- Bootstrap
- Axios
- React Router DOM

### Backend
- Node.js
- Express.js

### Database
- MongoDB
- Mongoose

## 📂 Project Structure

```
Frontend
│
├── User Registration
├── User Login
├── User Dashboard
└── Admin Dashboard

Backend
│
├── User Authentication APIs
├── Admin Authentication APIs
├── Feedback APIs
└── MongoDB Integration
```

## 🔄 Workflow

1. User registers and logs in.
2. User submits feedback or suggestions.
3. Feedback is stored in MongoDB.
4. Admin logs in and views all feedback.
5. Admin sends replies to feedback.
6. Users can view replies from the admin on their dashboard.
